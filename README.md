## ðŸ“¦ Installation

1.  Open **Extensions** in the sidebar of VS Code (`Cmd+Shift+X` or `Ctrl+Shift+X`).
2.  Search for `Nono Theme`.
3.  Click **Install**.
4.  Open the Command Palette (`Cmd+Shift+P` or `Ctrl+Shift+P`) and type `Preferences: Color Theme`.
5.  Select **Nono Theme Dark** or **Nono Light Theme**.
